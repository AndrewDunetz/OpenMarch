# musicxml-parser

A MusicXML parsing utility for the OpenMarch project

## Development

1. Ensure you have [Node](https://nodejs.org/en/download) installed
1. [pnpm](https://pnpm.io/installation) is optional but recommended
1. Install dependencies

   ```bash
   pnpm install
   ```

1. Run tests

   ```bash
   pnpm run test
   ```
