We want to hear from you! Your feedback helps make OpenMarch better for everyone.

## Where to Submit Feedback

Choose whatever works best for you:

- **[Discord](https://discord.openmarch.com)** - Quick questions and chat
- **[Facebook Group](https://facebook-group.openmarch.com)** - Community discussions
- **[GitHub Issues](https://github.com/OpenMarch/OpenMarch/issues)** - Bug reports and feature requests

We do read the Discord and the Facebook group, but we **submitting a GitHub issue** is the best way to ensure your feedback gets tracked and addressed.

## Why Use GitHub Issues?

GitHub issues are the best way to ensure your feedback gets tracked and addressed:

- We won't lose track of your request
- Other users can upvote or add to your idea
- You'll get notified when we fix or implement it
- Everything stays organized in one place
